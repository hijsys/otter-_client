2. Go to: Line 200 (around line 195-203)
3. Find this code:

       Enabled = true,
       Users = {
           ["Owner"] = {premium = true, expires = nil},
       },
       AdminKey = "OTTER_ADMIN_2024"

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

"Not whitelisted" error?
→ Check your Roblox username is correct in the whitelist
→ Username is case-sensitive!
→ Make sure you saved and uploaded the file

"404 Not Found" error?
→ Make sure you uploaded to GitHub first
→ Check the URL matches your username
→ Repository must be PUBLIC

GUI doesn't open?
→ Press Right Shift to toggle
→ Check if executor supports CoreGui

Settings don't save?
→ Some executors don't support writefile
→ Use manual save in Utility tab
